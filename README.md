### Привет!
Я Елена Першикова, тестировщик программного обеспечения.

### В мои обязанности входит:
- Анализ  документации, выявление недочётов в ТЗ, предложение решений
- создание тестовой документации: чек листы, тест-кейсы в confluence, отчеты о тестировании-проведение тест-дизайна
- работа в jira: создание баг репортов, заведение задач, логирование
- ручное тестирование ПО, регрессионное тестирование, смоук тестирование, тестирование сборки, интеграционное тестирование kafka через  swagger, postman (создание коллекций, mock сервера)
- оценка задач, участие в спринт планнингах, распределение задач между тестировщиками, решение возникающих вопросов, передача знаний, настройка стендов для тестирования
- работа в режиме многозадачности
- навыки работы с figma  и drawio, dev tools

### Автоматизация тестирования:
